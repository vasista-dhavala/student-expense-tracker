## Problem Statement
Students often spend money on food, travel, stationery, and small items throughout the day. Because these expenses are frequent and small, they often forget where their money is going. There is a need for a simple tool that lets students record their expenses easily.

## Scope of the Project
This project is a simple Java console application that allows users to:
- Add expenses
- View all expenses
- Delete expenses by ID
- Calculate total spending

The project focuses on Core Java, OOP concepts, and ArrayList implementation.

## Target Users
- Students who want to keep track of their daily spending
- Anyone who wants a lightweight expense tracker

## High-Level Features
- Add expense (name, category, amount)
- View expenses
- Delete expenses
- View total amount spent
- Simple menu-driven interaction
