# Student Expense Tracker

## Overview
Student Expense Tracker is a simple Java console-based application that helps students record, view, delete, and calculate their daily expenses. The project demonstrates Core Java concepts such as OOP, ArrayList, classes, objects, and basic CRUD functions.

## Features
- Add a new expense (name, category, amount)
- View all expenses
- Delete an expense using ID
- View total amount spent
- Simple and user-friendly console interface

## Technologies Used
- Java (Core Java)
- OOP Concepts
- Java Collections Framework (ArrayList)
- Scanner for user input

## How to Run
1. Install Java JDK.
2. Clone or download this repository.
3. Open the `src` folder in any Java-supported IDE (VS Code, IntelliJ, Eclipse).
4. Run the `Main.java` file.
